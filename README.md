🎮 Unity 3D Collection Game

A simple 3D game built using Unity, where the player navigates through obstacles and collects items to increase their score.

🚀 Features
Player movement with physics
Item collection system (score-based)
Basic obstacle interaction
UI score display (Count Text)
Simple 3D environment
🕹️ Gameplay
Control the player to move around the platform
Collect all the yellow items
Avoid obstacles and barriers
Score increases as you collect items
🛠️ Built With
Unity
C# scripting
📂 Project Structure
Assets/
 ├── Scripts/
 ├── Scenes/
 ├── Materials/
 ├── Prefabs/
