# 🎮 Runaway – Unity 3D Collection Game

A simple 3D game built using Unity where the player collects items while navigating through obstacles.

---

## 🚀 Features

- Player movement using physics  
- Item collection system (score-based)  
- Obstacles and collision interaction  
- UI score display  
- Simple 3D environment  

---

## 🕹️ Gameplay

- Move the player around the platform  
- Collect all yellow items  
- Avoid obstacles  
- Score increases as you collect items  

---

## 🎮 Controls

- **W / A / S / D** or **Arrow Keys** → Move player  

---

## 🛠️ Built With

- Unity (Game Engine)  
- C#
